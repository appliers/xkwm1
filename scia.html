SCIA (Under Maintence)

SCIA, also known as Changi International Airport IATA: SIN, ICAO: WSSS is a Discord bot programmed in v12 of Discord.js (Node.js) and holds a variety of commands
Invite your bot by clicking here

Commands



- !help
Show's Commands

- !version
Displays bots current version

- !invite
Sends an invite link

- !website
Sends a link to the website

- !A350
Sends a link to learn about the Airbus A350

- !777
Sends a link to learn about the Boeing 777

- !stats help
Tells you info about the stats commands

- !stats A350
Tells you information about the Airbus A350

- !stats 777
Tells you information about the Boeing 777

- ff
Replys with a pepe holding F (:pepeF:) (https://cdn.discordapp.com/emojis/752961636352000110.png?v=1)

- f
Replys your message with :pepeF:

- !kick
Kicks an user (!kick @user)

- !ban
Bans an user (!ban @user)
